# Counter GUI Application

## Overview
A simple Swing-based GUI application that implements a counter with increment and decrement functionality.

## Classes
- **ButtonApp**: Main application class extending JFrame that creates a window with two buttons for counting operations

## Features
- Increment counter with "++" button
- Decrement counter with "--" button
- Real-time display of counter value
- Clean and simple GUI interface

## Topics Covered
- Java Swing framework
- Event handling with ActionListener
- GUI components (JFrame, Button)
- Graphics rendering with paint() method
- Basic object-oriented programming

## How to Run
```bash
javac ButtonApp.java
java ButtonApp
```
