# Complex Numbers Calculator

## Overview
A Java application demonstrating complex number operations including addition and subtraction.

## Classes
- **complexnum**: Class representing complex numbers with real and imaginary parts
- **task1**: Utility class with static methods
- **Heloo**: Additional helper class

## Features
- Complex number addition
- Complex number subtraction
- Display complex numbers in standard form (a + bi)
- Object-oriented design for mathematical operations

## Topics Covered
- Object-oriented programming
- Class design and encapsulation
- Constructor overloading
- Method implementation
- Mathematical operations
- Object creation and manipulation
- Private fields and public methods

## How to Run
```bash
javac *.java
java [MainClassName]
```
