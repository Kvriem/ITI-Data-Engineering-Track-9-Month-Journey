# Student Management System

## Overview
A student information management system using ArrayList to store and display student records.

## Classes
- **student**: Model class representing student with personal and academic information
- **main**: Driver class demonstrating ArrayList operations and object management

## Features
- Store multiple student records
- Display formatted student information
- Dynamic collection management with ArrayList
- Encapsulated student data

## Topics Covered
- Class design and encapsulation
- ArrayList and Collections framework
- Constructor parameters
- Getter methods
- Object instantiation
- Iteration through collections
- Data modeling
- Information display formatting

## How to Run
```bash
javac *.java
java main
```
