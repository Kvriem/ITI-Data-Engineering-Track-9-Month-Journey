# Terminal File Editor

## Overview
A command-line file editor using FileInputStream and FileOutputStream for basic file read/write operations.

## Classes
- **TerminalFileEditor**: Main class providing terminal interface for file management

## Features
- Read file contents
- Write text to files
- Interactive terminal menu
- Multi-line text input
- Simple file management

## Topics Covered
- FileInputStream and FileOutputStream
- Byte array handling
- BufferedReader for console input
- Exception handling
- Try-with-resources for automatic resource management
- File I/O operations

## How to Run
```bash
javac TerminalFileEditor.java
java TerminalFileEditor
```
