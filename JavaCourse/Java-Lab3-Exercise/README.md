# Java Lab 3 Exercise

## Overview
Laboratory exercise demonstrating string manipulation and parsing operations with IP addresses.

## Classes
- **main**: Program that parses IP addresses into individual octets

## Features
- IP address parsing
- String splitting operations
- Array iteration
- Formatted console output
- User input handling

## Topics Covered
- Scanner class for input
- String manipulation methods
- split() method with regex
- Arrays and array indexing
- For loops and iteration
- String formatting
- Console I/O operations

## How to Run
```bash
javac main.java
java main
```
