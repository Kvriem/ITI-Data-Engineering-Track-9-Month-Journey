# Inheritance Demo

## Overview
A demonstration of inheritance in Java showing parent-child class relationships and method overriding.

## Classes
- **Parent**: Base class with basic arithmetic operations
- **Child**: Derived class extending Parent with additional functionality
- **Main**: Driver class demonstrating inheritance concepts

## Features
- Basic inheritance structure
- Method inheritance from parent to child
- Demonstrates "is-a" relationship
- Constructor chaining

## Topics Covered
- Class inheritance with extends keyword
- Constructor usage in inheritance
- Method inheritance
- Parent-child relationships
- Code reusability through inheritance
- Object-oriented design principles
- Super class and subclass concepts

## How to Run
```bash
javac *.java
java Main
```
