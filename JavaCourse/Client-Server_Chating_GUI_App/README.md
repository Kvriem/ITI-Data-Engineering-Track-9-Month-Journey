# Client-Server Chating GUI App

## Overview
This project demonstrates a simple client-server chat application using Java sockets. The server can handle multiple clients, broadcasting messages to all connected users. The client connects to the server, sends, and receives messages.

## Topics Covered & Learned
- Java networking (ServerSocket, Socket)
- Multithreading for handling multiple clients
- Input/output streams for communication
- Basic client-server architecture
