# Real-Time Clock GUI

## Overview
A Swing-based application that displays the current date and time, updating every second using multithreading.

## Classes
- **DateTime**: Main JFrame class that displays real-time date and time

## Features
- Real-time clock display
- Automatic updates every second
- Clean GUI interface
- Thread-based time updating

## Topics Covered
- Java Swing GUI
- Multithreading with Thread class
- Runnable interface implementation
- Date and time handling with Date class
- Graphics rendering
- Thread.sleep() for timing control

## How to Run
```bash
javac DateTime.java
java DateTime
```
