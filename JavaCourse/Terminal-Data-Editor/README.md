# Terminal Data Editor

## Overview
A terminal-based file editor that uses DataInputStream and DataOutputStream for reading and writing file contents.

## Classes
- **TerminalDataEditor**: Main class providing terminal interface for file operations

## Features
- Read file contents using DataInputStream
- Write text to files using DataOutputStream
- Interactive terminal menu
- Multi-line text input support

## Topics Covered
- Java I/O streams (DataInputStream, DataOutputStream)
- File handling operations
- BufferedReader for user input
- Exception handling with try-catch
- Resource management with try-with-resources

## How to Run
```bash
javac TerminalDataEditor.java
java TerminalDataEditor
```
