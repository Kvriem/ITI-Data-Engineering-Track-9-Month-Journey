# Interactive Line Drawing

## Overview
An interactive GUI application that allows users to draw lines by clicking and dragging the mouse.

## Classes
- **DrawLine**: Main JFrame class with mouse event handling for drawing

## Features
- Click and drag to draw lines
- Real-time line rendering
- Mouse event handling
- Dynamic graphics updates

## Topics Covered
- Java Swing framework
- Mouse event handling (MouseListener, MouseMotionListener)
- MouseAdapter and MouseMotionAdapter classes
- Graphics drawing methods
- Event-driven programming
- Real-time UI updates with repaint()

## How to Run
```bash
javac DrawLine.java
java DrawLine
```
