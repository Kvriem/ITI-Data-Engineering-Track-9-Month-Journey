# Scrolling Banner Animation

## Overview
An animated Swing application that displays a scrolling text banner moving from right to left.

## Classes
- **SimpleBanner**: JFrame application with animated scrolling text

## Features
- Horizontal text scrolling animation
- Smooth continuous movement
- Thread-based animation
- SwingUtilities for thread-safe UI updates
- Customizable message text

## Topics Covered
- Java Swing (JFrame, JLabel)
- Multithreading for animation
- Thread.sleep() for animation timing
- SwingUtilities.invokeLater() for thread safety
- Lambda expressions
- Component positioning with setBounds()
- Infinite animation loops

## How to Run
```bash
javac SimpleBanner.java
java SimpleBanner
```
