# Lamp Graphics Swing

## Overview
A Swing application that draws a decorative lamp using Java 2D graphics with polygons and ovals.

## Classes
- **Lamp**: Main JFrame class that sets up the window
- **DrawLamp**: Custom JPanel that handles the lamp drawing

## Features
- Custom graphics drawing
- Polygon shapes for lamp shade
- Multiple oval shapes for light bulbs
- Color customization
- Component-based design

## Topics Covered
- Java Swing (JFrame, JPanel)
- Graphics and Graphics2D
- paintComponent() method override
- Drawing polygons with fillPolygon() and drawPolygon()
- Drawing ovals with fillOval() and drawOval()
- Color class and custom colors
- Coordinate system and positioning

## How to Run
```bash
javac Lamp.java
java Lamp
```
