# Java Lab 2 Exercise

## Overview
Laboratory exercise focusing on conditional statements and user input handling.

## Classes
- **main**: Program demonstrating even/odd number checking with user interaction

## Features
- Interactive console application
- Number validation (even/odd)
- Loop for continuous checking
- User input handling
- Modulo operator demonstration

## Topics Covered
- Scanner class for user input
- Conditional statements (if-else)
- Boolean methods
- Modulo operator (%)
- Loops (do-while)
- String comparison
- Method design and separation of concerns
- User interaction flow

## How to Run
```bash
javac main.java
java main
```
