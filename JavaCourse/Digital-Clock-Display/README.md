# Digital Clock Display

## Overview
A Swing-based digital clock that displays the current time in HH:mm:ss format, updating every second.

## Classes
- **DateTimeDisplay**: JFrame application with threaded time updates

## Features
- Digital clock display
- HH:mm:ss time format
- Automatic updates every second
- Clean centered text display
- Custom font styling

## Topics Covered
- Java Swing (JFrame, JLabel)
- SimpleDateFormat for time formatting
- Multithreading with lambda expressions
- SwingConstants for alignment
- Font customization
- Thread.sleep() for timing
- Anonymous thread creation

## How to Run
```bash
javac DateTimeDisplay.java
java DateTimeDisplay
```
