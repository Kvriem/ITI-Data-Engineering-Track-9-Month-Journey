# Java Basics Practice

## Overview
A collection of basic Java programs for practicing fundamental programming concepts.

## Classes
- **Main**: Simple test program
- **test**: Additional practice exercises

## Features
- Basic program structure
- Console output
- Simple syntax practice

## Topics Covered
- Java program structure
- main() method
- System.out.println() for output
- Basic compilation and execution
- Package structure
- Class declaration

## How to Run
```bash
javac Main.java
java Main
```
